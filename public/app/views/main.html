Projects

  • Chat

    Description: Skaip is a chat program that you can download and chat with other online users.

    Nerdy info: Both the website and chat server are handwritten from scratch Java Servers. Client-server communication builds on top of the HTTP protocol. This was a great introduction in networking.

    Technologies used: HTTP, Java

  • Pixel Academy

    Description: Pixel Academy is a SPA that displays the staff and students of an academy. You can add, edit and delete users, so feel free to play with it.

    Nerdy info: The website is hosted by a written-from-scratch Java HTTP Server. The persons info is saved and retrieved in an SQL DB, via ajax calls. Object mapping uses JPA. This was an exercise on asynchronous calls and the challenges of Object-Relational Mapping.

    Technologies used: HTTP, Java, Javascript, JPA, MySQL, JSON

  • SeaCargo

    Description: SeaCargo is an imaginary transport company. You can view (or delete) info about its operations, so free to play (responsibly!) with it.

    Nerdy info: The website is hosted by a written-from-scratch Node Server. Client-server data transfer utilises ajax calls and JSON. Objects are saved in a noSQL DB (mongo). Html is build in real-time by server-side templating (Jade). This was an exercise on Node servers, noSQL, and server-side-templating.

    Technologies used: Node, Javascript, ajax, noSQL DB, JSON, Jade

  • GlobeJob

    Description: Globejob is site that gathers all the english-speaking job ads in Denmark. A useful tool for those who do not speak Danish.

    Nerdy info: My biggest project so far. The jobs are taken via web crawlers written in Java. The web server is a written-from-scratch Node server, while the client runs on Angular (html generated on run-time at client). This is a really distributed project, whose components run on 3 different cloud platforms.

    Technologies used: Node, Angular, Javascript, ajax, JSON, Java, JPA, mySQL
